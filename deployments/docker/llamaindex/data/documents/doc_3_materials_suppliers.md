## 2. Стальная арматура класса А500С

### Общая спецификация

- **Класс арматуры:** А500С (ГОСТ 52544-2006)
- **Диаметры используемые:** Ø 8, Ø 10, Ø 12, Ø 16, Ø 20, Ø 25, Ø 32, Ø 40 мм
- **Общий вес:** 8,500 тонн
- **Применение:** Основная рабочая арматура в монолитных конструкциях

### Поставщик: ОАО "НЛМК-Метиз"

| Параметр | Значение |
|---|---|
| **Базовая цена** | 42,500 р./тонна |
| **Контрактная цена** | 38,200 р./тонна |
| **Скидка объемная** | 10.1% |
| **Страна производства** | Россия (Липецкая область) |
| **Сертификация** | ГОСТ 52544-2006, ISO 6935 |
| **Сумма контракта (млн р.)** | 324.7 |

**Характеристики поставки:**
- Отгрузка: еженедельно по 50-70 тонн

- Доставка: автотранспортом, включена в стоимость контракта
- Контроль качества: испытания прочности каждой партии
