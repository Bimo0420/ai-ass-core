## 2. Стандартные пакеты отделки

### 2.1 Пакет "Эконом"

**Описание:** Базовая отделка жилого помещения

| Элемент отделки | Характеристики | Единица | Цена |
|---|---|---|---|
| **Потолки** | Гипсокартон, окраска | м² | 1,200 |
| **Полы** | Стяжка + ламинат | м² | 2,100 |
| **Стены** | Шпатлевка + окраска | м² | 850 |
| **Двери** | ДСП вход, ДВП внутренние | шт | 8,000 |
| **Окна** | Пластиковые стеклопакеты | шт | 12,000 |
| **Сантехника** | Унитаз (фаянс), раковина, ванна | компл | 35,000 |
| **Электрика** | Точечная установка розеток, выключателей | 1-комн | 15,000 |

**Средняя стоимость для 3-комн квартиры (85 м²):** **220,000 р.**

**Сроки выполнения:** 25-30 дней

### 2.2 Пакет "Комфорт"

**Описание:** Качественная отделка с современными материалами

| Элемент отделки | Характеристики | Цена за единицу |
|---|---|---|
| **Потолки** | Натяжные матовые | 2,500 р./м² |
| **Полы** | Стяжка + керамическая плитка (санузел), паркет (жилые) | 4,200 р./м² |
| **Стены** | Гипсокартон + финишная шпатлевка + краска (2 слоя) | 1,800 р./м² |
| **Двери** | Шпонированные двери (вход + внутренние) | 18,000 р./шт |
| **Окна** | Энергосберегающие стеклопакеты | 18,000 р./шт |
| **Сантехника** | Керамическая раковина, санфаянс премиум | 65,000 р./компл |
| **Электрика** | Скрытая электрика, встроенные светильники | 35,000 р./кв |

**Средняя стоимость для 1-комн квартиры (50 м²):** **380,000 р.**  
**Средняя стоимость для 3-комн квартиры (85 м²):** **580,000 р.**

**Сроки выполнения:** 45-50 дней

### 2.3 Пакет "Премиум"

**Описание:** Элитная отделка с дизайнерским подходом

| Элемент отделки | Характеристики | Цена за единицу |
|---|---|---|
| **Потолки** | Натяжные с подсветкой, декоративные покрытия | 4,500 р./м² |
| **Полы** | Мраморная плитка (холлы), паркет премиум (спальни), терраццо (ванная) | 6,800 р./м² |
| **Стены** | Венецианская штукатурка, обойные панели дизайнерские | 3,200 р./м² |
| **Двери** | Массив дерева, с фурнитурой премиум | 35,000 р./шт |
| **Окна** | Стеклопакеты энерго+звуко изоляция | 25,000 р./шт |
| **Сантехника** | Раковины из природного камня, биде, премиум смесители | 120,000 р./компл |
| **Электрика** | Автоматизация (затемнение, климат), встроенная музыка | 85,000 р./кв |

**Средняя стоимость для 1-комн квартиры (50 м²):** **750,000 р.**  
**Средняя стоимость для 3-комн квартиры (85 м²):** **1,100,000 р.**

**Сроки выполнения:** 60-70 дней
