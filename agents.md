# Задача: Создание документации для ai-ass-core

Ты - эксперт по технической документации. Необходимо создать полную документацию для этого проекта

## Контекст проекта

ai-ass-core - это локальная AI RAG agent система, объединяющая:
- Ollama (локальный LLM inference)
- n8n (автоматизация workflow)
- LlamaIndex (индексация и retrieval данных)
- Open WebUI (chat интерфейс)
- Supabase (база данных с pgvector)
- Docling (обработка документов)
- Traefik (reverse proxy и load balancing)
